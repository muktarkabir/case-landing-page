What is CASE?

CASE helps communities report local problems (infrastructure, safety, rights violations) and connects those reports to organizations that can help. Reports can be submitted through a web form, SMS, or USSD. Organizations claim cases, add verified updates, and create a public timeline so communities and funders can track real-world outcomes.

Multi-channel reporting: Web / SMS / USSD.

Evidence attachments: photos, short videos, timestamps.

Organization portal: claim cases, coordinate response, publish updates.

Public timelines: transparent, searchable case histories.

Privacy controls: anonymize reporter information when required.

How it works — 3 simple steps

1

Report

Submit a case via web, SMS or USSD. Add a short description, category, and optional photos. You’ll receive a reference ID.

2

Alert

CASE notifies verified local organizations and community leaders. Organizations can claim the case and post what they will do next.

3

Track

Follow a public timeline of actions and updates. If a case is stalled, community members can escalate to partner organizations or donors.

USSD (example):

Dial *123# → 1 Report an issue → Select category (e.g., Safety, Water, Health) → Enter short description → Receive reference ID via SMS.

SMS (example):

Text REPORT [location] [short description] to 08XXXXXX → We reply with a case ID and expected next steps.

Reporting via SMS and USSD is optimized for low-bandwidth and feature-phone users. Minimal required fields reduce friction.
